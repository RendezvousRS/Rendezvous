| 字符          | 全称（English）                                                             | 名称和类型（中文）                             | 常用 IO 格式说明符（printf / scanf）        |
| :------------ | :------------------------------------------------------------------------- | :--------------------------------------------- | :------------------------------------------- |
| int           | an integer, typically reflecting the natural size of integers              | ***整型 / 整数类型***                          | `%d`, `%i`                                   |
| short         | short integer                                                              | ***短整型***                                   | `%hd`, `%hi`                                 |
| long          | long integer                                                               | ***长整型***                                   | `%ld`, `%li`                                 |
| long long     | long long integer                                                          | ***长长整型***                                 | `%lld`, `%lli`                               |
| unsigned      | unsigned (integer modifier)                                                | ***无符号修饰符（配合 int/short/long 等）***  | 本身不单用：见 `unsigned int` 等            |
| unsigned int  | unsigned integer                                                           | ***无符号整型***                               | `%u`, `%o`, `%x`, `%X`                       |
| unsigned short| unsigned short integer                                                     | ***无符号短整型***                             | `%hu`, `%ho`, `%hx`, `%hX`                   |
| unsigned long | unsigned long integer                                                      | ***无符号长整型***                             | `%lu`, `%lo`, `%lx`, `%lX`                   |
| float         | single-precision floating-point type                                       | ***单精度浮点型***                             | `printf: %f`；`scanf: %f`                    |
| double        | double-precision floating-point type                                      | ***双精度浮点型***                             | `printf: %f`；`scanf: %lf`                   |
| long double   | extended-precision floating-point type                                     | ***长双精度浮点型***                           | `printf: %Lf`；`scanf: %Lf`                  |
| char          | a single byte, capable of holding one character in the local character set | ***字符型***                                   | `%c`                                         |
| signed char   | signed character type                                                      | ***有符号字符型***                             | `%hhd`（当作小整数用）                       |
| unsigned char | unsigned character type                                                    | ***无符号字符型***                             | `%hhu`（当作小整数用）                       |
| char[] / char*| array / pointer to char (C string)                                         | ***C 字符串***                                 | `%s`                                         |
| _Bool         | boolean type (since C99)                                                   | ***布尔类型（真 / 假）***                      | 一般作为 `int` 输出：`%d`                    |
| size_t        | unsigned integer type of `sizeof`                                         | ***用于表示大小的无符号整型***                 | `%zu`                                        |
| ptrdiff_t     | signed integer type of pointer difference                                  | ***指针差值的有符号整型***                     | `%td`                                        |
| wchar_t       | wide character type                                                        | ***宽字符类型***                               | `%lc`（单个宽字符），`%ls`（宽字符串）       |
| void          | (no value) indicates absence of type or return value                       | ***空类型***：无返回值 / 无参数                 | 不能直接 IO；见下一行的 pointer / `%p`      |
| void *        | pointer to void                                                            | ***void 指针类型***                            | `%p`                                         |
| pointer (T *) | pointer to type T                                                          | ***指针类型***                                 | `%p`（以实现定义方式打印指针值）             |
| nothing       | (no corresponding data type; only used by `%n`)                            | ***无对应数据类型，仅用于记录“字符个数”***     | `%n`（把目前已读/已写的字符数存入变量）      |
