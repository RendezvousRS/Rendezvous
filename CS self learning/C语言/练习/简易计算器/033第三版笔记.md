# 第三次修改前作的笔记

#### 缺点

>- 1、不能随意加减乘除，只能连加、连减、连乘、连除。
>- 2、没有用到数组。根本意义是没有练习到数组的使用。

#### 033第二版的错题

>- 1、***scanf*** 输成 sanf
>- 2、; 打成中文格式 ；
>- 3、***main(){}*** 不符合新标准，需改成 ***int main(void){}***
>- 4、***double sum(void)*** 要在main函数前定义
>- 5、xcode报错提示“不能定义这个函数（在乘除法函数那里）

# 第三次修改时遇到的问题

#### 格式本身

>- 1、有的函数后面缺了一个大括号 }
>- 2、减法那里少了***return j;***

#### 语言

>- 1、减法函数里，双精度浮点数格式输入的数字在接收时定义的先验值未赋值 纠正后为***double j =0.0;***
