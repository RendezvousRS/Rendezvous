## 无限个数输入

gpt 提供了一种思路

引入布尔值

```
#include <stdio.h>

int main(void) {
    double x;
    double sum = 0.0;

    printf("请输入任意多个数字，以空格或回车分隔。\n");
    printf("输入结束后，Linux/macOS 用 Ctrl+D；Windows 用 Ctrl+Z 再回车结束输入。\n");

    while (scanf("%lf", &x) == 1) {
        sum += x;   // 每读到一个，就累加到 sum 里
    }

    printf("这些数的和是：%lf\n", sum);
    return 0;
}
```
